Description

This 'status' property (format 10) is created specifically for use by the Articles module.

On input, the 'status' property provides a drop down list of possible status options for an article.

On output display the 'statu's property provide the named of the status level. eg Frontpage, Published, Draft, and so on, as defined in the articles_userapi_getstates() function.

Attributes

There are no available attributes or configuration options for this property.
